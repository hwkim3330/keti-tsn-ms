Traffic Class Shaper Configuration

About Traffic Class Shapers

CBS (Credit-Based Shaper): IEEE 802.1Qav - Guarantees bandwidth for AVB traffic using idle-slope parameter.

SLB (Single Leaky Bucket): Standard token bucket shaper using CIR (Committed Information Rate) and CBS (Committed Burst Size).

Important: Each traffic class can only have ONE shaper mode. Clear existing config before switching modes.
